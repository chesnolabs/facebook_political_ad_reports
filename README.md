# facebook_political_ad_reports

Аналіз даних Facebook про витрати на рекламу, пов'язану із соціальними питаннями, виборами або політикою.
Дані взято звідси: https://www.facebook.com/ads/library/report/?source=archive-landing-page&country=UA#

## Як це працює:
Оскільки витрати менше 100 доларів Facebook не деталізує (а на коротких проміжках це можуть бути невеликі суми), ми щомісяця зберігаємо звіт за весь час відстеження витрат (з 2019 року). Таким чином, якщо за весь час сторінкою було витрачено більше 100 доларів, ми можемо відстежувати навіть дрібні витрати.

### Приклад:
Минулого місяця сума витрат за весь час становила 102 долари, цього місяця – 107 доларів. Значить, за місяць ця сторінка витратила на рекламу 5 доларів.

## Структура:

20nn (2023, 2024, ...) - теки по роках з архівами сирих файлів звітів Facebook, по одному файлу за останній день кожного місяця. Для поточного місяця файли завантажуються та зберігаються щодня. На початку наступного місяця всі файли за минулий місяць, окрім останнього, видаляються.

monthly_data_merged.csv - файл з агрегованими даними в дфох форматах:
- абсолютні суми з накопиченням, на останній день кожного місяця
- витрати за місяць (різниця витрат за останній день місяця та за останній день попереднього місяця)

monthly_data_merged.xlsx - той самий файл, але у форматі Excel
